# agile-ai-skills

This repository stores skills for the Agile AI app.

It supplements the app by providing reusable skill definitions and examples, and it serves as a central repository for the Agile AI skills marketplace.

## Purpose

- Store and organize skills used by the Agile AI application
- Support marketplace-style discovery and reuse of skill definitions
- Keep skill content separate from app code so the Agile AI app can consume and extend skills easily

## Usage

Use this repository as the source of truth for Agile AI skills that can be published, shared, and updated independently of the main application.
